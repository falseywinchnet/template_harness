# Preserved sources and data

Store original external inputs here before normalization: papers, datasets,
instrument exports, correspondence supplied for the project, and source
snapshots permitted by their licenses. Organize each source in its own directory
with a short provenance record.

Recommended record:

```text
ID: E001
Origin/author:
Acquired at:
Stable locator:
License/access constraints:
Original filename:
SHA-256:
Transformations: none
Claims using it:
```

External URLs are locators, not substitutes for preserved evidence. Do not commit
restricted, private, export-controlled, or identifying data merely because the
template expects local evidence; document the controlled location and access
procedure instead.
